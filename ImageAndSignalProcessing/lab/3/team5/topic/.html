1. Contrast Stretching

The Theory

Low-contrast images can result from poor illumination, lack of dynamic range in the image sensor, or even wrong setting of a lens aperture during image acquisition.

The idea behind contrast stretching is to increase the dynamic range of the grey levels in the image being processed. In the case of contrast stretching, the output value of the pixel depends only on the input value of the same pixel (contrary to convolution where the output pixel value depends on a whole neighborhood of the corresponding input pixel). These processes are generally called LUT for Look Up Table.

The following Figure shows the result of contrast stretching on an MRI slice.

Figure 1.1. Application of contrast stretching on an MRI slice.

Let us consider the actual range of grey levels of the input signal being the interval [actualMin,actualMax] and the final range of grey levels being [newRangeMin,newRangeMax]. Generally, the final range is wider than the original one when we want to spread contrasts.

The process of contrast stretching is a non-causal system which can be described as:

y[n]=newRangeMin+newRangeMax−newRangeMinactualMax−actualMin(x[n]−actualMin)

def read_pgm(filename, byteorder='>'):
    """Return image data from a raw PGM file as numpy array.
 
    Format specification: http://netpbm.sourceforge.net/doc/pgm.html
 
    """
    with open(filename, 'rb') as f:
        buffer = f.read()
    try:
        header, width, height, maxval = re.search(
            b"(^P5\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n])*"
            b"(\d+)\s(?:\s*#.*[\r\n]\s)*)", buffer).groups()
    except AttributeError:
        raise ValueError("Not a raw PGM file: '%s'" % filename)
    return np.frombuffer(buffer,
                            dtype='u1' if int(maxval) < 256 else byteorder+'u2',
                            count=int(width)*int(height),
                            offset=len(header)
                            ).reshape((int(height), int(width)))
