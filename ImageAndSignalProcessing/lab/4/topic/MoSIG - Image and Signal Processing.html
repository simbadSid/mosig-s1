For images, generally, the 2D Fourier Transform is centered around zero. This means that instead of considering the Fourier Transform from n=0 to n=N−1, we consider it from n=−N2+1 to n=N2. This is possible as we can consider a finite length signal as an infinitely long periodic signal.

This can be obtained by symmetry. Indeed, |F(f)[−u,−v]|=|F(f)[u,v]| as F(f)[−u,−v]¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯¯=F(f)[u,v].

def Ellipse(outputTableShape, center2DCoord, radius2D, insideFlag) :
#Return a 2D array with the size :
#outputTableShape[0] = number of columns (Ny)
#outputTableShape[1] = number of lines (Nx)
#The center ofthe circle is center2DCoord (xc, yc)  
#The radius is 2D radius[0] for a, and 2Dradius[1] for b  
#The equation of the ellipse is ((x-xc)/a)^2 + ((y-yc)/b)^2 = 1  
#If insideFlag is true, the value of output is 1 inside the ellipse and 0 outside 
#If insideFlag is false, the value of output is 0 inside the ellipse and 1 outside 
    
    Nx = outputTableShape[1] 
    Ny = outputTableShape[0] 
       
    xc = center2DCoord[0] 
    yc = center2DCoord[1]  
   
    a = radius2D[0] 
    b = radius2D[1]  
   
    x = np.linspace(0, Nx-1, Nx) 
    y = np.linspace(0, Ny-1, Ny) 
    x2DCoord, y2DCoord = np.meshgrid(x, y)
          
    ellipse = np.zeros(outputTableShape) 
    xScale = (x2DCoord-xc)/a 
    yScale = (y2DCoord-yc)/b 
       
    ellipse = (xScale)**2 + (yScale)**2 
       
    if insideFlag: 
        segmentEllipse = ellipse <= 1 #type boolean 
    else: 
        segmentEllipse = ellipse > 1 #type boolean 
      
    segmentEllipse = 1*segmentEllipse #type integer
       
    return segmentEllipse
