def myMin(x):
    m = x[0]
    for k in range(len(x)):
        if (x[k] < m):
            m = x[k]       
    return m
     
def myMax(x):
    m = x[0]
    for k in range(0, len(x)):
        if (x[k] > m):
            m = x[k]       
    return m
        

The function myMin takes a signal (i.e. an array) as input and returns its minimum value.

6. Random Signal

The mathematical expectation of a continuous random process, called the average of the process is defined as:

E[χ]=∫∞−∞xdF(x)=∫∞−∞xp(x)dx
where F is the Probability Distribution Function and p is the Probability Density Function of the random variable. (see Definition I.5 in the lecture notes page 21).

Question 6.1

The theoretical standard deviation of a random variable is given by

σχ=E[X2]−(E[X])2−−−−−−−−−−−−−√
